# 前后端接口文档

## 登入接口
### 请求URL
```
/api/login
```
### 请求方式
```
POST
```
### 请求参数
| 参数名   | 类型   | 是否必须 | 备注   |
| -------- | ------ | -------- | ------ |
| userName | string | yes      | 用户名 |
| password | string | no       | 密码   |
### 返回数据
| 参数名  | 类型    | 是否必须 | 备注           |
| ------- | ------- | -------- | -------------- |
| status  | boolean | yes      | 登入的返回状态 |
| message | string  | yes      | 返回信息       |
